# 更新日志

所有重要的项目变更都会记录在这个文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，

## [1.0.0] - 2026-02-10

### 🎉 首次发布

首个正式版本，包含 10 个核心飞书 API Skills。

### ✨ 新增

- **10 个核心 Skills**：
  - `feishu-im` - 消息与群管理（25+ API）
  - `feishu-doc-writer` - 文档写入（15+ API）
  - `feishu-drive` - 云空间文件管理（10 API）
  - `feishu-bitable` - 多维表格（30+ API）
  - `feishu-task` - 任务管理（5 API）
  - `feishu-calendar` - 日历日程（5 API）
  - `feishu-approval` - 审批流（5 API）
  - `feishu-contact` - 通讯录（10 API）
  - `feishu-wiki` - 知识库（5 API）
  - `feishu-card` - 交互卡片（10 API）

- **完整文档**：
  - 每个 Skill 包含详细的 API 说明
  - 100+ 条实测心法和避坑指南
  - 典型应用场景和最佳实践

- **开源配套**：
  - MIT 开源协议
  - 完整的贡献指南
  - 快速开始脚本

### 📝 特性

- **即插即用**：Markdown 格式，AI Agent 直接读取
- **跨框架支持**：支持 Cursor、Claude 等任何 LLM 框架
- **实战验证**：所有 API 均经过实际项目验证
- **持续更新**：随飞书 API 更新而更新

### 🔒 安全

- 所有敏感信息已脱敏
- 无真实用户数据
- 包含权限安全指南

### 📚 文档

- README.md - 项目主文档
- LICENSE - MIT 开源协议
- CONTRIBUTING.md - 贡献指南
- CHANGELOG.md - 更新日志
